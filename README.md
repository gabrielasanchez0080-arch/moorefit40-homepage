# MooreFIT 40+ Homepage

This is a minimal Next.js (App Router) + Tailwind starter with your live homepage.

## Quick Start
1) `npm install`
2) `npm run dev` (visit http://localhost:3000)
3) `npm run build && npm start` for production

## Deploy to Vercel
- Push to GitHub and import in Vercel, or drag-and-drop the folder to Vercel's dashboard.
